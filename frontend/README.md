# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/eec6fb16-751d-48a7-8e6e-bc332d2270a2

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Configure Firebase client auth env vars in `.env` (see `.env.example`).
3. Run the app:
   `npm run dev`
